# 📜 UPSC — TestBook

> A Testbook-style UPSC Civil Services Prelims mock test platform built for serious aspirants. Practice with real PYQs, get UPSC-accurate scoring, and track your preparation with rich analytics.

---

## 🖥️ Live Demo

> [Practice the PYQs](https://upsc-test-book.vercel.app/)

---

## ✨ Features

### 🧪 Mock Test Engine
- **Previous Year Questions (PYQs)** from 2012–2025 structured in clean JSON
- **Year-wise practice mode** — attempt papers from a specific year
- **Section-wise practice mode** — drill down on History, Geography, Polity, Economy, Environment, Science & Tech, and more
- **UPSC-accurate negative marking** — +2 for correct, −0.66 for wrong, 0 for unattempted
- **Timed exam simulation** — replicates real Prelims pressure

### 🧭 Question Navigator
- Live status panel showing Answered / Unanswered / Marked for Review
- Jump to any question instantly
- Mark questions for review before final submission

### 📊 Scorecard & Analytics
- Section-wise score breakdown
- Strength & weakness detection across subjects
- Difficulty-level performance (Easy / Medium / Hard)
- If needed check the explanations on google via direct google search on every questions
- Attempt accuracy and net score summary

### 💾 Test History (Local Storage)
- **Every test attempt is automatically saved to your browser's Local Storage** — no login or account required
- Resume where you left off if you accidentally close the tab
- Review past test attempts, scores, and answers anytime
- All data stays on your device — completely private

### 🧠 Vision IAS Powered Explanations
- **Every question comes with a thorough, detailed explanation** sourced and inspired by **Vision IAS** study material
- Explanations don't just tell you the correct answer — they explain *why*, connecting concepts to the broader UPSC syllabus
- Ideal for building conceptual clarity, not just rote memorisation

### 🎨 UI & Design
- Deep navy and saffron/amber color scheme — inspired by the official UPSC aesthetic
- Serif typography for a formal, exam-hall feel
- Fully responsive — works on desktop and mobile

## 🚀 Getting Started

### Prerequisites
- Node.js ≥ 16
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/mailmeatdarshan/UPSC---TestBook.git

# Navigate into the project
cd UPSC---TestBook

# Install dependencies
npm install

# Start the development server
npm start
```

The app will be available at `http://localhost:3000`.

### Build for Production

```bash
npm run build
```

## 📖 How to Use

1. **Select a mode** — Choose Year-wise or Section-wise practice
2. **Start the test** — Questions load with a countdown timer
3. **Navigate freely** — Use the Question Navigator panel to jump around
4. **Mark for review** — Flag uncertain questions to revisit before submitting
5. **Submit & analyse** — View your scorecard with section-level breakdown and performance insights
6. **Review explanations** — Read Vision IAS-based explanations for every question after submission
7. **Pick up later** — Your test progress is auto-saved in Local Storage so nothing is ever lost
